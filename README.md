# 🏢 StayNest - Premium Housing Marketplace

**Verified Student Housing & Brokerage-Free Rentals**

StayNest is a world-class, premium web application built entirely with Python and Streamlit. It serves as a trusted ecosystem connecting students, working professionals, property owners, and universities—all without charging any brokerage fees. 

The platform utilizes heavy custom CSS and HTML injections to provide a premium SaaS experience reminiscent of Airbnb and Stripe, bypassing the default dashboard look of Streamlit.

## 🚀 Features

* **Premium UI/UX:** Glassmorphism, smooth animations, and responsive rounded cards.
* **Role-Based Access Control:** Dedicated dashboards for Students, Owners, Universities, and Admins.
* **Smart Search & Filters:** Advanced filtering by budget, amenities, distance, and university proximity.
* **AI Recommendation Engine:** Smart property matching based on lifestyle and budget.
* **Advanced Analytics:** Real-time Plotly dashboards for platform health and revenue statistics.
* **100% Brokerage-Free:** Direct connections with verified owners.

## 🛠️ Tech Stack

* **Frontend:** Streamlit, Custom HTML/CSS
* **Backend:** Python 3.10+
* **Database:** SQLite (Architected for easy PostgreSQL migration)
* **Data Visualization:** Plotly & Pandas
* **Security:** Bcrypt (Password Hashing)
